ConanVimrc v 0.1

For more info please read vimrc file in this folder

1. clone the project to your `Home` directory (or simply download the zip/tar file, and extract to your `Home` directory)
$ mkdir ~/.vim_runtime
$ git clone git@github.com:ConanChou/ConanVimrc.git ~/.vim_runtime

2. Install
$ cat ~/.vim_runtime/install.sh
" read through the configs if you want
$ sh ~/.vim_runtime/install.sh <system>
" <sytem> can be `mac`, `linux` or `windows`
